# MVP to Launch Roadmap

## MVP

- Reality-based character creation.
- Fate analysis: zodiac, five elements, temperament, fortune, talent.
- Random job and trait generation with hidden jobs.
- Beginner dungeon random room chain.
- Text RPG choices, turn combat, loot, quests, enhancement, local autosave.
- OpenAI story endpoint with local fallback.
- Prisma schema for PostgreSQL production persistence.

## Beta

- Server-authoritative dungeon and combat resolution.
- PostgreSQL save/load in `/api/game/save`.
- NextAuth social providers and account linking.
- Guild, warehouse, crafting recipes, daily reset scheduler.
- Job-specific skill trees with unlock choices.
- Equipment set bonuses and market sale pricing.
- AI NPC dialogue memory with moderation and output schema validation.

## Launch

- Seasonal dungeon tables and boss rotations.
- Reincarnation leaderboards.
- Paid cosmetic-only supporter packs.
- Vercel production deployment with managed PostgreSQL.
- Observability: error tracking, event analytics, economy dashboards.
- Content pipeline for new rooms, monsters, item affixes, and quests.
